## TLOS is listed on various Decentralized Exchanges (DEX):

Typically, anyone can list a coin/token on a DEX.

:::caution
Whenever you're trading on a DEX, make sure to check if the contract address is correct since there might be fake clones made by scammers to deceive people.  
Telos Zero: eosio.token  
Telos EVM: 0xD102cE6A4dB07D247fcc28F366A623Df0938CA9E  
Ethereum: 0x7825e833d495f3d1c28872415a4aee339d26ac88  
Binance Smart Chain (BSC): 0xb6c53431608e626ac81a9776ac3e999c5556717c  
Telos does not exist on other blockchains, if you see a coin on another chain claiming to be TLOS it is a scam
:::

**For an up to date list of current dexes please refer to our ecosystem [DeFi page](https://www.telos.net/ecosystem-apps-filter/defi?tab=evm)**
