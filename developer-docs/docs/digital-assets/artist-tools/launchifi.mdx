# Launchifi.xyz

## Overview
Launchifi is a no code solution that allows you to deploy contracts straight to the blockchain! Luckily for us here in the Telos Digital Asset department that also includes contracts such as ERC-721 and ERC-1155.

You can visit Launchifi using the link below!

[Launchifi - Launch Anythin to the Blockchain | A Web3 Launchpad](https://launchifi.xyz/)
